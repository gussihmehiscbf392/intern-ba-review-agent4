# Источники 2026

Материалы скопированы из `D:\стажировка 2026` для локальной и воспроизводимой работы профиля.

Используемые файлы:

- `Описание кейса стажировка.docx` -> контекст кейса (`materials/case.md`)
- `интервью к кейсу.docx` -> факты интервью (`materials/interview.md`)
- `Шаблон ФТТ.docx` -> ожидаемая структура ФТТ (`materials/ftt_template.md`)
- `Инструкция_к_Шаблону_ФТТ.docx` -> правила заполнения (`materials/template_instruction.md`)
- `Инструкция_к_Шаблону_ФТТ_2026_обновление.docx` -> актуализированные правила заполнения (используется в `materials/template_instruction.md` с 28.04.2026)
- `Задание №1_Сбор и формализация требований.docx` -> постановка задания Week 1
- `Онлайн курс в рамках программы отбора младших специалистов.docx` -> контекст программы
- `Критерии оценки.docx` -> актуальная шкала оценки (обновление от 27.04.2026), используется в `materials/criteria_2026.md`
- `ФТТ_стажировка_пример_хорошая_работа.docx` -> референс хорошей работы (не эталон), используется для калибровки эвристик
- `kriterii_2026_priemka_mvp_agenta_ba.docx` -> историческая версия шкалы 2026 и критерии приемки MVP (`materials/mvp_acceptance_2026.md`)
