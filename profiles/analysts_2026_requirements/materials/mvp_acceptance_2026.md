# Критерии приемки MVP-агента (2026)

Источник: `kriterii_2026_priemka_mvp_agenta_ba.docx`.

## Что должно быть выполнено

- Загрузка одного ответа стажера в формате `DOCX/PDF`.
- Автоподбор внутреннего пакета материалов (кейс, интервью, шаблон, инструкция, критерии).
- Формирование структурированного отчета (резюме, критерии, примеры несоответствий, обратная связь).
- Объяснимость замечаний через фрагменты текста или указание отсутствующего смысла.
- Корректный тон: направление на исправление без автодорешивания задания за стажера.
- Работа без эталонов решенных заданий как обязательного условия.
- Поддержка подключения калибровочных примеров (strong/medium/weak).
- Устойчивость к ошибкам входных файлов и корректные сообщения об ошибках.
- Повторяемость результатов при повторной проверке одного и того же документа.
- Готовность к подключению OpenAI API через отдельный слой вызова модели.

## Что не входит в MVP

- Проверка других заданий аналитиков.
- Проверка потоков инженеров данных.
- Полноценная админ-панель.
- Массовая пакетная проверка десятков работ.
- Интеграции с YT, базой знаний и корпоративными системами.
