# Калибровочные примеры (опционально)

Система работает без этой папки и без примеров.
Если примеры есть, они используются только для калибровки строгости уровней.

## Формат файла

Создавайте `*.json` файлы со схемой:

```json
{
  "id": "example_001",
  "level": "strong",
  "overall_score": 88.5,
  "criterion_scores": {
    "completeness": 90,
    "placement": 85,
    "quality": 87
  },
  "mentor_comment": "Краткий комментарий наставника"
}
```

Где:

- `level`: `strong` | `medium` | `weak`
- `overall_score`: итоговый балл примера
